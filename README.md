# 🎛️ Discord Bot Music V2
Bot musik Discord premium menggunakan `discord.js` v14, `discord-player` v6, dan didukung oleh FFmpeg untuk pemrosesan audio filter secara *real-time*.

## Feature
- 🎧 High-quality audio streaming (YouTube, Spotify VVIP, SoundCloud fallback)
- 🎛️ Interactive Audio Filter Manager (/filter)
- 📊 Live Status Dashboard
